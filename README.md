# Static Website Deployment (Practice Project)

## Overview
This is a **static website practice project** created and deployed to understand basic website structure and deployment workflow.

## Pages
- Home (index.html)
- About (about.html)
- Products (product.html)
- Blog (blog.html)
- Single Post (singlepost.html)
- Contact (contact.html)

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Deployment
- Static website
- No backend or database
- Deployed for practice purposes
- Can be hosted on NGINX, Apache, or GitHub Pages

## Purpose
Learning static website structure, navigation, and deployment basics.
